# Strategy

This is a low-frequency, aggressive-growth rollout strategy for a small dedicated Robinhood Agentic Account.
It is designed for scheduled checks every 30 minutes, not for fast stop-losses, event-chasing, or intraday scalping.
The goal is higher upside capture through carefully gated small positions, not uncontrolled trade frequency.

Universe:
- Premarket source universe: `config/universe.txt`.
- Daily trade allowlist: `state/today_allowlist.txt`, generated by the premarket agent.
- Daily watchlist: `state/daily_plan.json.today_watchlist`.
- Intraday may trade only the intersection of those files.

Priority themes for the premarket screen:
- AI semiconductors and compute.
- AI data-center infrastructure, networking, power, and cooling.
- CPO, photonics, interconnect, and narrow-gate supply-chain candidates.
- Space, defense, drones, satellites, and autonomy.
- Nuclear, uranium, power, and energy infrastructure.
- Broad beta ETFs only when single-name conditions are weak or market risk is elevated.

General:
- Only act when `state/daily_plan.json` allows the action.
- Only act on symbols in `state/today_allowlist.txt` and today's `today_watchlist`.
- Only act in the dedicated Robinhood Agentic Account.
- Prefer high-conviction small orders over broad low-quality activity.
- If there are no strong setups, do nothing.

Daily dynamic allowlist construction:
- Select 3-8 symbols from `config/universe.txt`.
- Rank sectors first, then symbols.
- Score each candidate 0-100:
  - theme momentum 20
  - catalyst/news quality 20
  - technical confirmation 20
  - liquidity/tradability 15
  - portfolio/account fit 15
  - downside/volatility control 10
- 80-100: tradable candidate if risk rules pass.
- 65-79: watch only unless price confirms strongly.
- Below 65: do not include in today's trade allowlist.
- Block symbols with unverified catalysts, earnings gap risk, liquidity concerns, unclear tradability, or stale quote data.

Buy rule:
- Only if `daily_plan.allowed_actions` includes `small_limit_buy`.
- Only if the symbol is in `state/today_allowlist.txt` and `today_watchlist`.
- For broad ETFs, prefer pullbacks: current price down at least 0.5%-1.0% from prior close or reclaiming intraday VWAP after weakness.
- For high-momentum single names, allow breakout/reclaim setups only if premarket plan marks `breakout_allowed=true` for that symbol and quote data is fresh.
- Do not buy if `market_regime` is `risk_off` or `no_trade`.
- Do not buy if the account already has an open order for that symbol.
- Do not average down automatically if the existing position is below cost basis.
- Limit price must be at or below the latest verified quote.
- Order notional must respect `config/risk.md`, `config/runtime.env`, and `state/daily_usage.json`.

Sell rule:
- Only sell existing long positions in the Agentic Account.
- Only if `daily_plan.allowed_actions` includes `partial_take_profit`.
- If a position is up more than 2.5% from cost basis, consider partial take profit.
- If a position is down more than 3%, do not sell automatically; log an alert only.
- Never short.
- Limit price must be at or above the latest verified quote unless explicitly reducing risk after a review-safe take-profit setup.

Default:
- If unclear, do nothing.
- If data is missing, stale, conflicting, or hard to reconcile, do nothing.
- If no high-quality setup appears, append a `no_action` decision with the reason.
