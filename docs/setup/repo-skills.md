# Repo Skills Setup

## Install

```bash
./scripts/install_repo_skills.sh
./scripts/verify_repo_skills.sh
```

## Source of Truth

- Repo source: `.agents/skills/`
- Installed copies:
  - `$HOME/.agents/skills`
  - `~/.codex/skills`

## Notes

- Installs are real copies, not symlinks.
- Re-run the installer after repo skill updates.
