# Buy and Sell Point Taxonomy

## Type 1

Type 1 is the earliest reversal point. It appears when a trend-ending move shows divergence away from the last center.

Use it when:

- the old trend looks exhausted
- force weakens into the last extreme
- you want the earliest reversal attempt

Risk:

- highest false-positive rate
- often needs fast execution and tight discipline

## Type 2

Type 2 is the retest after the first reversal leg. It is less early but usually cleaner.

Use it when:

- type 1 has already created a first reversal response
- the retest cannot extend the old extreme with authority
- you want better confirmation than type 1

## Type 3

Type 3 is the continuation setup after price leaves a center, pulls back, and does not re-enter the center.

Use it when:

- the market has already declared a directional bias
- you want continuation, not bottom-picking
- the pullback respects the old center boundary

## Priority of Use

- Type 1 for early reversal
- Type 2 for more confirmed reversal
- Type 3 for cleaner continuation

## Sell Logic

The exact same taxonomy applies on the short side:

- type 1 sell: exhaustion into a final high
- type 2 sell: retest after the first down leg
- type 3 sell: failure to re-enter the old center after breakdown

## Common Misreads

- calling a weak bounce inside damaged structure a type 2 buy
- calling a move a type 3 buy even though price has re-entered the center
- treating every divergence as a tradable type 1
