# Grade Selection

## Core Rule

Choose one execution level before analysis. Do not switch levels just because the current position feels uncomfortable.

## What Each Level Does

- Higher level: defines the dominant state and the meaningful center boundaries.
- Execution level: defines the trade thesis and valid buy or sell points.
- Lower level: refines timing only after the higher-level structure is already valid.

## Selection Framework

Use one level based on holding period and tolerance for noise:

- intraday: minute-based execution levels
- swing: hourly or daily execution levels
- position: daily or weekly execution levels

## Non-Mixing Rule

Common failure:

- buy on a 5-minute structure
- panic because a 1-minute pullback looks weak
- then justify the hold using the daily chart

This is a level mix. The trade loses discipline because entry, fear, and thesis all live on different clocks.

## Practical Test

Before calling a signal, answer:

1. What is the execution level?
2. What center or trend leg exists on that level?
3. What exact event invalidates the thesis on that same level?

If any answer requires changing levels, the read is unstable.

## Upgrade Logic

A small-level trade may evolve into a larger-level position only if:

- the higher level is also supportive
- the move has clearly expanded into higher-level trend structure
- you explicitly change the trade plan and stop logic

Do not "accidentally" upgrade a trade because it moved in your favor.

## Common Errors

- using lower-level noise to negate a higher-level third buy
- calling a higher-level reversal based only on lower-level divergence
- changing execution level after entry to avoid taking a loss
