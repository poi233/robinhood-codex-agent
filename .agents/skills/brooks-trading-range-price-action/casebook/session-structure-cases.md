# Session Structure Cases

## Case 1: Morning Breakout, Midday Downgrade

Context:
- open drives hard in one direction
- midday overlap expands

Correct interpretation:
- early trend, later range

Operational takeaway:
- session regime can change before the day ends

## Case 2: Range Open, Afternoon Resolution

Context:
- early two-sided trading
- later breakout gains real follow-through

Correct interpretation:
- breakout mode at the open, clearer trend state later

Operational takeaway:
- wait for better information when the first hour is noisy
